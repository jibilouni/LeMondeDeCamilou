# Images pour Le Monde de Camilou

Ce dossier contient les images utilisées sur le site web.

## Images à ajouter

Veuillez placer les images suivantes dans ce dossier :

### Services
- `yoga.jpg` - Photo pour le service Yoga
- `ongles.jpg` - Photo pour le service Prothésie Ongulaire
- `esthetique.jpg` - Photo pour le service Esthéticienne

### Ateliers
- `atelier-1.jpg` - Photo d'atelier créatif
- `atelier-2.jpg` - Photo d'atelier créatif
- `atelier-3.jpg` - Photo d'atelier créatif
- `atelier-4.jpg` - Photo d'atelier créatif

### Optionnel
- `hero.jpg` - Image de fond pour la section hero (si vous souhaitez remplacer le gradient)

## Format recommandé

- **Format** : JPG ou PNG
- **Dimensions minimales** : 1200px de largeur
- **Poids** : Optimisées pour le web (< 500KB par image)
- **Aspect ratio** : 16:9 pour les services, carré pour les ateliers

## Placeholder

En l'absence d'images, le site affiche automatiquement un placeholder avec un dégradé de couleurs pastel et le texte "Photo à venir".
